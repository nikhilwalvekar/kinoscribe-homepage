Learn

KinoScribe is a 3D Presentation creation tool. It is based on HTML5 Web Components. Hence it is super easy to use. All you have to do is to import the Web Component on your page and start using it.

More over KinoScribe is meant to be extensible. If you have a great idea about an Cool Transition or Layout or Any Tools around Presentation, go ahead and write your own Custom Element. Your Custom Element or JavaScript code can interact with KinoScribe using Attributes and Events
